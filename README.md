
# Responsi PAML
Mobile App using Flutter Framework

## Tech Stack
**Front-End:** Flutter, Dart


## Run Locally

Clone the project

```bash
  git clone https://github.com/X3nonxe/responsi_PAML.git
```

Go to the project directory

```bash
  cd responsi
```

Install dependencies

```bash
  flutter pub get
```

Start the project

```bash
  flutter run
```
